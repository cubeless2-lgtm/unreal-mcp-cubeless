Verifier note:

- Dynamic material preview graphs can produce incomplete generated point metadata after the spawner, even when final ISM output and material slot behavior are correct.
- The ecosystem verifier therefore relaxes the material preview component's point metadata check only for `dynamic_actor_property` graph mode.
- Style/tree graph paths, style/tree point counts, and disjoint material ISM counts are still checked normally.

## Implementation Notes

The material axis is currently integrated as a separate PCG component. It previews and validates material override semantics next to the ecosystem output, but it does not yet dynamically override the materials of the style/tree components generated by the same actor.

That separation avoids duplicating all style/tree graph combinations while preserving a verified material override control surface.
